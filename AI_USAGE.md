# AI_USAGE.md

## Was AI used in this project?
Yes.

AI was used as a support tool during development, planning, code generation, debugging, and interface improvement.

---

## What AI was used for?
AI was used for:
- brainstorming project structure
- planning page flow
- generating initial Streamlit code
- improving UI guidance and usability
- refining cleaning and visualization logic
- helping prepare project documentation

---

## What was verified manually?
The following items were verified manually:

- the app runs locally in Streamlit
- file upload works for CSV
- overview metrics display correctly
- duplicate detection/removal works
- type conversion works
- scaling functions work
- visualization builder produces charts
- export buttons generate output files
- UI changes and guidance text were checked visually
- app behavior was tested on sample datasets

---

## What still required human judgment?
Human decisions were required for:
- choosing datasets
- deciding which UI layout was clearer for users
- checking whether charts were meaningful or misleading
- evaluating whether the app matched assignment requirements
- reviewing explanations and documentation

---

## Important note
AI assistance was treated as a development aid, not as a substitute for testing or human evaluation. All major features were reviewed and tested manually before inclusion.
